# python爬虫测试练习

##第一个python爬虫的小项目

准备做一个收集马蜂窝中游记的爬虫，收集一个地点的所有游记
的文字内容，然后通过对文字内容进行语义情感分析，得出各个
地点的好感程度。

test中是些测试的小例子，requests/selenium等测试

### -----

目前以三亚为例，通过城市页面，使用游记的ajax接口，获取所有游记的链接
通过各个链接访问单独的游记页面，下载游记内容(单线程,单代理)。

目前爬取速度有问题，太快会被封ip

完成一个简单的离线proxy获取工具，后期可使用别的动态代理池维护工具
